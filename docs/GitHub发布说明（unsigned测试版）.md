# GitHub 发布说明（unsigned 测试版）

## 推荐标题

`oh fans 1.0 (build 1) unsigned 测试版`

## 推荐正文

`oh fans` 是一个面向 Apple Silicon Mac 的菜单栏风扇控制工具。

当前版本是未签名测试版，适合通过 GitHub Release 做小范围分发验证。

### 本次发布包含

- 菜单栏常驻风扇控制面板
- 主窗口和设置页
- Apple Silicon 风扇读取与控制
- app 内辅助控件安装 / 重装链路
- DMG 内附“如果打不开请看这里”README

### 适用范围

- 仅支持 Apple Silicon（M1 / M2 / M3 / M4 及后续 M 系列）
- 不支持 Intel Mac

### 下载内容

- `oh-fans-1.0-1-unsigned.dmg`

### 安装方式

1. 下载并打开 DMG。
2. 把 `oh fans.app` 拖到 `Applications`。
3. 进入“应用程序”目录，优先右键 `oh fans.app` -> “打开”。
4. 首次进入后，如果界面提示安装辅助控件，按提示输入管理员密码完成安装。

### 重要说明

- 这是 unsigned 测试版，没有 Apple Developer 签名和 notarization。
- macOS 首次打开时，可能提示“无法验证开发者”“已损坏”或直接打不开。
- 如果遇到这类情况，请先看 DMG 里的 `README_如果打不开请看这里.md`。
- README 里已经写了右键打开、“隐私与安全性”里的“仍要打开”，以及 `xattr` 处理命令。

### 当前边界

- 当前优先保证真实控制链路可用，还不是正式签名分发版本。
- 辅助控件二进制或通信协议变更后，需要重新安装辅助控件。
- 如果你只想先验证 app 能不能启动，请至少先完成一次辅助控件安装，再确认是否退出只读监控模式。

### 反馈时请尽量带上这些信息

- 你的 Mac 型号和芯片型号
- macOS 版本
- 是否成功打开 app
- 是否成功安装辅助控件
- 模式切换后风扇是否真的变化
- 如果失败，失败弹窗原文或截图
